# 📈 Bitcoin Market Sentiment Analysis

## 📌 Project Overview

This project analyzes the relationship between **Bitcoin Market
Sentiment (Fear & Greed Index)** and **historical trader performance**
to identify hidden patterns and generate actionable trading insights.

## 🎯 Objective

Analyze how Bitcoin market sentiment influences trader profitability and
identify actionable insights for improving trading strategies.

## 📈 Statistical Tests

-   One-Way ANOVA (Closed PnL vs Market Sentiment)
-   One-Way ANOVA (Trade Size vs Market Sentiment)
-   Pearson Correlation (Trade Size vs Closed PnL)
-   Chi-Square Test (Profit Flag vs Market Sentiment)

------------------------------------------------------------------------

## 🔍 Key Findings

-   Market sentiment significantly affects trader profitability.
-   Fear generated the highest trading activity.
-   Extreme Greed achieved the highest average profitability.
-   Trade size has only a weak positive relationship with profitability.
-   Winning rates varied across market sentiment categories.
-   Trading activity followed distinct hourly and weekly patterns.
